# 🌌 MoodReads — AI-Powered Book Discovery

> *Find books that match your vibe.*

A full-stack book discovery platform with semantic AI recommendations, mood journaling, AI-generated summaries, and a personal reading shelf — built with FastAPI, React, and Claude.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Mood-Based Discovery** | Pick a mood → Claude semantically matches books to your emotional state |
| **Mood Journal** | Write how you feel → AI analyzes your text and picks books |
| **AI Summaries** | Spoiler-free 2-minute summaries generated by Claude |
| **AI Librarian** | Chat with Claude about any book or ask for personalized recs |
| **Reading Shelf** | Track Want to Read / Reading / Finished / Maybe Later |
| **Vibe Ratings** | Rate books on Comfort, Adventure, Romance, Tears, Mind-blown |
| **25 Pre-seeded Books** | Curated collection across moods and genres |

---

## 🛠 Tech Stack

**Backend** · FastAPI · SQLAlchemy · SQLite (local) / PostgreSQL (prod) · JWT Auth · Anthropic Claude API

**Frontend** · React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion · React Router v6

## 🚀 Running Locally

### Prerequisites

- Python 3.10+
- Node.js 18+
- An Anthropic API key → https://console.anthropic.com

---

### Step 1 — Backend

```bash
cd moodreads/backend

# Create a virtual environment
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set up environment
cp .env.example .env
```

Open `.env` and paste your Anthropic API key:
```
ANTHROPIC_API_KEY=sk-ant-...
```

```bash
# Seed the database with 25 books
python seed_data.py

# Start the API server
uvicorn app.main:app --reload --port 8000
```

API is now running at **http://localhost:8000**
Swagger docs at **http://localhost:8000/api/docs**

---

### Step 2 — Frontend

Open a second terminal:

```bash
cd moodreads/frontend

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open **http://localhost:5173** in your browser.

## 🌐 Deploying to Render (Free Tier)

### Backend

1. Push to GitHub
2. New Web Service on Render → connect repo → set root dir to `moodreads/backend`
3. Build command: `pip install -r requirements.txt && python seed_data.py`
4. Start command: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`
5. Add environment variables:
   - `DATABASE_URL` → use the Render PostgreSQL connection string
   - `ANTHROPIC_API_KEY` → your key
   - `SECRET_KEY` → a long random string

For PostgreSQL: also `pip install psycopg2-binary` — add to requirements.txt

### Frontend

1. New Static Site on Render → connect repo → root dir `moodreads/frontend`
2. Build command: `npm install && npm run build`
3. Publish dir: `dist`
4. Add env var: `VITE_API_URL=https://your-backend-url.onrender.com`
5. Update `vite.config.ts` proxy to use `import.meta.env.VITE_API_URL`

---

## 🔑 Environment Variables

| Variable | Default | Description |
|---|---|---|
| `DATABASE_URL` | `sqlite:///./moodreads.db` | SQLite locally, PostgreSQL on Render |
| `SECRET_KEY` | dev key | Change this in production |
| `ANTHROPIC_API_KEY` | *(empty)* | Required for AI features |
| `ACCESS_TOKEN_EXPIRE_MINUTES` | `43200` (30 days) | JWT expiry |

---

## 🤖 AI Features (without API key)

All AI features degrade gracefully — the app still works, but:
- Mood recommendations fall back to tag-based filtering
- Summaries, journal analysis, and librarian chat show a "configure API key" message

## 📝 Adding More Books

Edit `seed_data.py` and add entries to the `BOOKS` list. Delete `moodreads.db` then re-run `python seed_data.py`.

Valid moods: `Happy, Sad, Heartbroken, Motivated, Dreamy, Peaceful, Adventurous, Thrilling, Romantic, Mystery`
